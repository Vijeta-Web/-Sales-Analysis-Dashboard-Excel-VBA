# -Sales-Analysis-Dashboard-Excel-VBA

📌 Project Overview

* The Sales Analysis Dashboard is an interactive Excel-based reporting solution designed to analyze and monitor sales performance across multiple cities and sales executives. The dashboard provides dynamic insights into total sales, target achievement percentage, and performance trends using advanced Excel features and VBA automation.

-- This project focuses on improving reporting efficiency, reducing manual effort, and enabling data-driven decision-making.

🚀 Key Features

* The dashboard was built using Pivot Tables, Pivot Charts, Slicers, Filters, Conditional Formatting, and VBA Macros to create a fully interactive reporting environment.

// It allows users to dynamically filter data city-wise and evaluate:

- Total Sales by Sales Executive

- Target Achieved Percentage

- Target Not Achieved Analysis

- Sales Performance Trends

- Executive-wise Sales Distribution

- The automation implemented through VBA significantly reduces repetitive tasks and ensures faster report generation.

⚙️ Tools & Technologies Used

- Microsoft Excel

- Pivot Tables & Pivot Charts

- Slicers & Filters

- Conditional Formatting

- VBA (Visual Basic for Applications)

- Macros Automation

📈 Business Impact

* The dashboard automated over 15+ weekly manual reporting tasks, reducing time consumption and improving operational efficiency. It enhanced reporting accuracy and enabled quick performance tracking at both executive and regional levels. The automation and structured design improved decision-making speed and minimized dependency on manual data processing.

🧠 Insights Generated

* The dashboard helps in identifying:

1- Top and bottom performing sales executives

2- Regions with high or low target achievement

3- Sales trends across different cities

4- Performance gaps requiring management attention

📂 How to Use

* Download the Excel file from this repository.

-- Enable Macros when prompted.

-- Use the slicers at the top to filter data by city.

-- Analyze performance using the interactive charts and summary tables.

🎯 Project Objective

* The primary goal of this project was to create an automated, interactive, and user-friendly sales reporting solution that reduces manual workload and provides actionable business insights.
